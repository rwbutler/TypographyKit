## Does this project accept contributions?

Absolutely! See the section below on how to get started.

## How can I contribute?

- For discussion use: 
	1. Check if an open issue already exists on [GitHub Issues](https://github.com/rwbutler/TypographyKit/issues).
	2. If not, create a new issue [using the relevant form](https://github.com/rwbutler/TypographyKit/issues/new/choose).
- To contribute code: 
	1. Get agreement by raising an issue using [GitHub Issues](https://github.com/rwbutler/TypographyKit/issues).
	2. Create a [PR](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request) linking to the original issue that was raised.
